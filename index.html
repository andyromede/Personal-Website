Movie Query September 2021

Lunar Landing Novemeber 2021

  • This project is based off a game called Lunar Landing which involves navigating the explorer back to his shuttle (middle of the grid) with the help of bumping into helper bots in its vicinity.
  • Uses MVC model and implemented smart BFS backtracking solver to give user hints to solve the puzzle.
  • Shows mastery of MVC model, Java, Object Oriented Programming, multifile programs, and attention to user experience.
  • GitHub link: https://github.com/andyromede/bfspuzzles-Lunar_Landing_and_Tip_Over-callbackcats
